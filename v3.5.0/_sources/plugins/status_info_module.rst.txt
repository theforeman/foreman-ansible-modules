.. note::
    This module is part of the `theforeman.foreman collection <https://galaxy.ansible.com/theforeman/foreman>`_ (version 3.5.0).

    see :ref:`Requirements <ansible_collections.theforeman.foreman.status_info_module_requirements>` for details.
