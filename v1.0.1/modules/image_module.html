image – Manage Images¶

  • Synopsis

  • Requirements

  • Parameters

  • Examples

  • Status

Synopsis¶

  • Create, update, and delete Images

Requirements¶

The below requirements are needed on the host that executes this module.

  • apypie

Parameters¶

Parameter Choices/Defaults Comments
architecture
string / required
architecture of the image
compute_resource
string / required
Compute resource the image is assigned to
image_password
string
Password that is used to login into the operating system
image_username
string / required
Username that is used to login into the operating system
name
string / required
Image name
operatingsystem
string / required
Operating system that will be deployed using the image
password
string / required
Password of the user accessing the Foreman server

string / required
URL of the Foreman server
state
string
    Choices:
  • present ←
  • absent
State of the entity

boolean
    Choices:
  • no
  • yes
Image supports user_data
username
string / required
Username accessing the Foreman server
uuid
string / required
UUID or Marketplace URN of the operatingsystem image

validate_certs
boolean
    Choices:
  • no

Whether or not to verify the TLS certificates of the Foreman server

Examples¶

- name: create Image for EC2
  image:
     name: CentOS
     image_uuid: "ami-0ff760d16d9497662"
     image_username: "centos"
     operatingsystem: "CentOS 7"
     compute_resource: "AWS"
     architecture: "x86_64"
