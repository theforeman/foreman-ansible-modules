.. Title

theforeman.foreman.scc_product module -- Subscribe SUSE Customer Center Account Products
++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++

.. Collection note

.. note::
    This module is part of the `theforeman.foreman collection <https://galaxy.ansible.com/theforeman/foreman>`_ (version 3.4.0-dev).

    You might already have this collection installed if you are using the ``ansible`` package.
    It is not included in ``ansible-core``.
    To check whether it is installed, run :code:`ansible-galaxy collection list`.

    To install it, use: :code:`ansible-galaxy collection install theforeman.foreman`.

    To use it in a playbook, specify: :code:`theforeman.foreman.scc_product`.

Synopsis

.. Description

- Manage SUSE Customer Center Products
- This module requires the foreman_scc_manager plugin set up in the server
- See \ https://github.com/ATIX-AG/foreman_scc_manager\
